# Smart Notes Hub 📝

Smart Notes Hub is a modern note-taking web application developed using React for the frontend and Firebase for backend services. It helps users create, manage, and organize notes easily with a simple and user-friendly interface.

## 🚀 Features

- Add new notes
- Edit existing notes
- Delete notes
- Responsive user interface
- Real-time data storage using Firebase
- Clean and simple design
- Google Fonts integration for better UI appearance
- Fast performance using React + Vite

## 🛠️ Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript
- React.js
- Bootstrap
- Vite

### Backend
- Firebase

### Additional Tools
- Google Fonts
- Font Awesome Icons
